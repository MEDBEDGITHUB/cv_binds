## How to bind a new gestureğŸ‘Œ
1. Show gesture that you want to bind
2. Press enter
3. Write in console name of the gesture and button or hotkey you want to bind
4. Show hand again
5. Press enter
6. Remove hand
7. Show hand again
8. Press enter
Your gesture is saved
